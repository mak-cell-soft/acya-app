# 🚀 Premium Next.js Boilerplate

A professional, high-performance starter kit for building modern web applications with Next.js 15+, Tailwind CSS 4, and the best-in-class tools.

## 🛠 Tech Stack

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) + [Shadcn UI](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Data Fetching:** [TanStack Query v5](https://tanstack.com/query/latest) + [Axios](https://axios-http.com/)
- **Forms & Validation:** [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
- **Server Actions:** [Next-Safe-Action](https://next-safe-action.dev/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Notifications:** [Sonner](https://sonner.stevenly.all/)

## 📁 Project Structure

- `src/app`: Next.js App Router (Pages, Layouts, API)
- `src/components/ui`: Shadcn UI components (Radix primitives)
- `src/components/shared`: Reusable business components
- `src/hooks`: Custom React hooks
- `src/lib`: Utilities, shared constants, and action clients
- `src/services`: Server Actions and API logic
- `src/store`: Zustand global state definitions
- `src/types`: TypeScript interfaces and types

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
